## 4. What Version 0.1 does not prove

Version 0.1 does not yet prove:

- that the curriculum improves learner outcomes;
- that the capability model is complete;
- that the evidence model is valid for high-stakes use;
- that the framework works across all contexts;
- that learners using it will outperform others;
- that the future conditions are correct predictions;
- that the curriculum should replace formal schooling;
- that all included content is optimally sequenced;
- that all excluded content is unimportant.

Revise this list after Stage 11.

## 12. Revision triggers

Version 0.1 should be revised if:

- important evidence contradicts a core claim;
- implementation feedback reveals serious usability issues;
- AI conditions change materially;
- safeguarding concerns emerge;
- a capability is shown to be poorly defined;
- an evidence route proves invalid or burdensome;
- public users misunderstand a core claim;
- traceability gaps are found.
