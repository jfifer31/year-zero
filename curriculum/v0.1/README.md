# Year Zero Education – Version 0.1 Curriculum

This folder is the release container for the first public curriculum hypothesis produced by the Year Zero Education research chain.

## Status

**Current status:** Structural scaffold only

**Curriculum content authorised:** No

**Next authorising stage:** Stage 9 for evidence-of-learning architecture; Stage 10 for curriculum and implementation content

**Final release gate:** Stage 11

Stage 0 creates the files and neutral structures needed by later stages. It does not authorise curriculum principles, domains, capabilities, pathways, assessment models, learning experiences or implementation recommendations.

## Files in this folder

| File | Distinct purpose | Earliest authorising stage |
|---|---|---|
| `README.md` | Release-container status and navigation | Stage 0 |
| `CURRICULUM_FRAMEWORK.md` | Curriculum aims, principles, domains, pathways and learning architecture | Stage 10 |
| `EVIDENCE_OF_LEARNING.md` | Evidence, validity, portfolio and verification architecture | Stage 9 |
| `IMPLEMENTATION_GUIDE.md` | Guidance for parents, educators, mentors and learners | Stage 10 |
| `LIMITATIONS_AND_OPEN_QUESTIONS.md` | Release limitations, unresolved questions and revision triggers | Stage 10; finalised Stage 11 |
| `TRACEABILITY_MAP.md` | Links from release elements to authorised upstream records | Stage 10; reviewed Stage 11 |

## Population rule

A later stage may populate a section only when its passed outputs authorise the content. Candidate examples, prompt language and attractive ideas are not evidence and must not be promoted into the release by default.

Every major release element must be traceable through:

```text
source or evidence
→ classified claim
→ stage finding
→ authorised pass-forward statement
→ curriculum or evidence element
```

## Required release standard

Version 0.1 may be published only after Stage 11 confirms that it is traceable, appropriately cautious, publicly understandable, implementation-aware and safe enough for its stated use.

Version 0.1 must not claim proven effectiveness, formal qualification status, universal suitability, prediction accuracy or replacement of statutory education.
